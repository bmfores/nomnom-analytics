## Introduction

Every day, numerous events take place across the university campus, often with catering. When these events end, leftover food is up for grabs. NomNom Analytics analyzes WhatsApp group data, where students post details of events with leftover food, to help students efficiently locate and consume this food.

## Features

- Post Frequency Analysis
- Location Hotspots
- Food Types Analysis
- Food Depletion Time Analysis
- Poster Leaderboard
- Post Quality vs. Depletion Time
- Predictive Modeling
- Time between Posts and Depletion
- Visual Recognition Insights
- Food Preferences Analysis

## Data Collection

We scrape the following data from the WhatsApp group:

- Name of the person who posted
- Date and time of the post
- Day of the week
- Room/location
- Attached pictures
- Type of food served
- Time taken for the food to be depleted after posting
